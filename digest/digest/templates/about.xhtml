Project information

The goal of this web application is to perform an enzymatic digest of a user-provided protein sequence. The user will interact with the program via a TurboGears web-interface and will be able to specify different parameters (specified below) for the digest. The output will be a table of peptides with their length, molecular weight, # of missed cleavages, and the amino-acids to left and right of each peptide in the protein sequence

Parameter information

  • Protein sequence (UniProt id or text)
    • Can not contain spaces in the middle of the sequence
    • Input needs to at least 6 characters long
    • For accessing UniProt, this program will only consider the main protein sequence and cannot access variants or isoforms.
  • Enzyme
    • Supported Enzymes are listed here
  • min and max length of the fragment
  • min and max weight of the fragment
    • This program uses the molecular weight of an amino acid. Does not consider charge.
    • Modifications to methionine (such as oxidation) or cysteines (such as reductions) are not considered.
    • PTMs are not considered in mass calculation
    • 18 Da is added to all fragments to account for rehydration of the fragments after protein cleaving.
  • number of missed cleavages allowed (0-5)

Output

A table of fragments, of valid length and weight, with the following information.
  • Number of missed cleavages
  • The fragment sequence
  • The length of the fragment
  • The weight of the fragment
  • The amino acid to the left and right of the splice site

Programmatic Access

You can access the program with the following urls, depending on what output you would like. Paramaters should be joined together with the & symbol.

  • Normal: http://0.0.0.0:8080/digest?Parameters
  • XML: http://0.0.0.0:8080/digest.xml?Parameters
Parameters
  • query - Protein sequence or UniProtID to be digested
  • enzyme - default - Trypsin
  • min_l - default=0
  • max_l - default = 10000
  • min_w - default = 0
  • max_w - default = 100000
  • misses - default = 0
